I HAVE TAKEN AIRLINE MANAGEMENT AS MY MINI PROJECT WHEREIN THE CODE OFFERS THE PASSENGER(S) TO BOOK THEIR ITIENARY. THE PASSENGER IS GIVEN WITH THE OPTIONS OF CHOOSING THE DEPARTING CITY AND THE ARRIVAL CITY FROM A VARIETY OF CITIES OFFERED(VIZ MUMBAI, KOLKATA, DELHI..) THE PASSENGER HAS THE FREEDOM OF CHOOSING THE DAY OF DEPARTURE AND ALSO THE PREFERRED TIMING. AFTER PROVIDING THE PREFERENCES THE CODE OFFERS THE OPTIONS SUITING THE PASSENGER'S CHOICES. ALSO THE FOLLOWING DETAILS ARE ASKED FROM THE USER TO GENERATE THE TICKET:
1. NAME
2. ADDRESS
3. GENDER
4. AGE
5. CONTACT INFORMATION
6. NUMBER OF PASSENGERS
7. PAYMENT MODE
	A. CASH 
	B. CREDIT/DEBIT CARD

THE CODE GIVES THE USER THE FREEDOM TO CHOOSE THE CLASS TO TRAVEL IN(ECONOMY/BUSINESS), THE SEAT(WINDOW/MIDDLE/AISLE), PREFERRED MEAL(VEG/NON VEG) AND THE BAGGAGE.
